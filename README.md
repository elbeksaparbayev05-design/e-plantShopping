# e-plantShopping

e-plantShopping – bu onlayn o‘simlik do‘koni. Foydalanuvchilar turli kategoriyadagi o‘simliklarni ko‘rishlari, tanlashlari va savatga qo‘shishlari mumkin.

## Features
- Plant categories: Indoor, Outdoor, Flowering
- Dynamic product list with name, thumbnail, and price
- Add to Cart functionality with cart counter
- Checkout and continue shopping buttons

## Installation
1. Clone the repository
2. Run `npm install`
3. Start the project with `npm start`
